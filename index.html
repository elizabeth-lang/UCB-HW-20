UX/UI Portfolio Case Study

The Prompt: Create a website using HTML, CSS, JQuery, and Bootstrap My Role: Junior UX/UI Designer Tools Used: Figma, Canva, VSCode, GitHub, Adobe Color

Phase 1: Ideation

Content

Define introduction, skills, work, and navigation content.

Moodboard

Find visual inspiration by researching existing portfolio websites.

Phase 2: Drafting

First Draft

Used VSCode and GitHub to create a navigation bar, hero image, skills section with text and imagery, and a work section displaying a snippet of my case studies, with a button to learn more. Using JQuery, I created a smooth scrolling UX interaction to make the page functional. I created a color palette using Adobe Color and applied it using CSS.

Phase 3: Testing

Goal: Test smooth scrolling

I tested smooth scrolling by clicking on a menu item in both the header and the footer to observe correct function. Success!

Goal: Get feedback on visual design

Feedback received: change logo, modify weight of button font, change button border color.

Phase 4: Solution

Solution

1. Added new logo 2. Changed button font weight 3. Changed button border color
